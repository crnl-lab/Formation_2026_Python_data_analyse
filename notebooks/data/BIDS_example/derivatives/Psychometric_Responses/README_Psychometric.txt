
README for Psychometric_Responses Folder

This folder includes psychometric questionnaire scores for each participant, collected prior to the EEG task.
Variables include: PANAS, Openness/Intellect, Curiosity, VVIQ, AVIQ, Mindfulness, AReA, and demographics.

Participant ID convention is as follows:
	•P101 = Participant 1  
	•P102 = Participant 2  
	•	….
	•P151 = Participant 51  
This ID convention is used consistently across all EEG, behavioural, and psychometric data files.

Measured Variables:

•PANAS – Positive and Negative Affect Schedule (Start and End)
•Openness/Intellect – 20 items
(Items 1–10: Openness; Items 11–20: Intellect)
•Curiosity
•VVIQ – Vividness in Visual Imagery 
•AVIQ – Vividness in Auditory Imagery (BAIS-V, renamed as AVIQ)
•Mindfulness – From the MAAS scale
•AReA –Aesthetic Responsiveness 
•Demographics:
	•Demo_1 – Gender
	•Demo_2 – Age
	•Demo_3 – Ethnicity
	•Demo_4 – Location
	•Demo_5 – Education
	•Demo_6 – Poetry Liking
	•Demo_7 – Prior exposure to poetry

Questionnaires.pdf
	•Provides the text references for all questionnaires used
	•Includes scoring keys and scale interpretations
