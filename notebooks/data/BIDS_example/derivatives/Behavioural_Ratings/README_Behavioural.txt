
README for Behavioural_Ratingsfolder

This folder contains behavioural response data for each participant, recorded during the poem evaluation task. Each CSV file corresponds to one participant (e.g., P101.csv) and includes ratings on five subjective dimensions: Aesthetic Appeal, Vivid Imagery, Being Moved, Originality, and Creativity (1–7 scale).
Variables:
•PoemName – First line of the poem or text
•PoemType – Experimental condition (H = Haiku, S = Senryu, C = Control)
•Block – Block structure of trials (1–7)
•Subjective ratings:
	•AA – Aesthetic Appeal
	•Imagery – Vivid Imagery
	•Moved – State of Being Moved
	•Originality
	•Creativity

Participant ID convention is as follows:
	•P101 = Participant 1  
	•P102 = Participant 2  
	•	….
	•P151 = Participant 51  
This ID convention is used consistently across all EEG, behavioural, and psychometric data files.
